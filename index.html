📘 Getting Started

Getting Started

Download the SnapUML library and include it in your project. 

<script src="snapuml.min.js"></script>

Once loaded, the window.snapuml global object becomes available.

🚀 Manual Rendering

Use render() to manually convert diagram code to SVG. This function automatically detects the diagram type (Sequence or Component). 

const code = 'Alice -> Bob: Hello'; // or component diagram code
const svg = snapuml.render(code);

// Insert into DOM
document.getElementById('output').innerHTML = svg;

Parameters:

• content: SnapUML-format diagram code

Returns:

SVG string. On error, returns styled error SVG (no exceptions thrown).

🔄 Auto-Rendering

Call initialize() to auto-detect and render all diagram blocks. Similar to Mermaid. 

snapuml.initialize({
  startOnLoad: true,
  selector: 'pre.snapuml'
});

// Or use defaults:
snapuml.initialize();

Config Options:

• startOnLoad: Auto-render on page load (default: true)

• selector: CSS selector (default: 'pre.snapuml')

Note:

Finds all matching elements, extracts text content, and replaces with SVG diagrams.

📄 HTML Usage Example

Code:

<html lang="en">
  <head>
    <meta charset="utf-8" />
  </head>
  <body>
    <h1>Diagrams</h1>

    <h2>Example 1</h2>
    <pre class="snapuml">
Alice -> Bob: Hello!
Bob -> Alice: Hi!
    </pre>

    <h2>Example 2</h2>
    <pre class="snapuml">
participant C
participant S

C -> S: Request
S -> C: Response
    </pre>

    <script src="snapuml.min.js"></script>
    <script>
      snapuml.initialize();
    </script>
  </body>
</html>

Rendered Output:

Example 1

Alice -> Bob: Hello!
Bob -> Alice: Hi!

Example 2
