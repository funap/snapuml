# SnapUML

A PlantUML-compatible diagram generator for the web.

**[Try it Live](https://funap.github.io/snapuml/)**

## Build Instructions

### Prerequisites
- Node.js (v18.0.0 or later recommended)
- npm

### Installation

```bash
npm install
```

### Build

```bash
npm run build
```

This generates the following files in the `dist/` directory:
- `index.js` - Browser bundle
- `index.min.js` - Minified browser bundle

### Type Check

```bash
npm run type-check
```

### Run Tests

```bash
npm run test
```
